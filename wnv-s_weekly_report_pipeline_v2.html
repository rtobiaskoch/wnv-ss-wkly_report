Weekly Data Report

DATA Prep checklist

  1. Change Boulder week to epiweek
  2. check that all files have been uploaded to the googledrive
  3. check the quanstudio eds files on diomni pcr software on thermofisher and look to see if any inconclusives with a Ct are positive or negative. If any are negative add them to the amp_inconclusive_negative database sheet in the google drive
  4. download the entire wXX folder from googledrive into the input folder in this repository
